# Design Brainstorm - Liga Tucuruí CS2

## Chosen Design: **Esports Premium Dark**

### Design Movement
Modern esports aesthetic with luxury gaming sensibilities. Inspired by professional CS2 tournament broadcasts and high-end gaming interfaces.

### Core Principles
1. **Dark Elegance**: Deep charcoal/black backgrounds with gold accents for premium feel
2. **Competitive Energy**: Dynamic layouts that convey intensity and performance
3. **Data Clarity**: Clean typography and structured information hierarchy for easy scanning
4. **Cinematic Depth**: Layered visuals with subtle animations and depth effects

### Color Philosophy
- **Primary**: Deep black (`#0a0e27`) - authority and focus
- **Accent**: Gold (`#d4af37`) - prestige, victory, ranking
- **Secondary**: Deep blue (`#1a2f5a`) - trust, competitive spirit
- **Text**: Off-white (`#f0f0f0`) - readability on dark backgrounds
- **Highlights**: Neon cyan (`#00d9ff`) - energy, modern gaming

### Layout Paradigm
- **Hero Section**: Full-width cinematic image with overlay gradient
- **Asymmetric Grid**: Left-aligned content with right-side accent elements
- **Modular Cards**: Pots displayed as premium card containers with hover effects
- **Vertical Flow**: Hero → Standings → Pots (progressive revelation)

### Signature Elements
1. **Gold Dividers**: Horizontal lines separating sections with gradient fade
2. **Rank Badges**: Visual indicators for pot levels (star, diamond, skull icons)
3. **Glow Effects**: Subtle neon glows on interactive elements and highlights

### Interaction Philosophy
- Smooth transitions on all interactive elements
- Hover states reveal additional information
- Cards lift on hover (shadow depth increase)
- Smooth scroll animations for section reveals

### Animation
- Fade-in animations on page load
- Subtle scale transforms on card hover
- Gradient animations on accent elements
- Smooth scroll-triggered reveals for data sections

### Typography System
- **Display Font**: Oswald (bold, condensed) - headlines, section titles
- **Body Font**: Inter (clean, modern) - descriptions, data
- **Monospace**: Courier New - player stats, ratings
- **Hierarchy**: Large display (48px) → Medium (28px) → Small (16px)
